# 阶段关键代码

> 记录时间：2026-06-20 | 阶段：Phase 0 — 项目初始化

## 1. pyproject.toml（依赖配置）

```toml
[project]
name = "rance-blog"
version = "0.1.0"
requires-python = ">=3.14"
dependencies = ["django>=5.2,<6.0"]
```

- 使用 Django 5.2 LTS 系列
- Python 最低 3.14

## 2. manage.py（Django 入口）

由 `django-admin startproject config .` 自动生成，位于项目根目录。

```python
#!/usr/bin/env python
"""Django's command-line utility for administrative tasks."""
import os
import sys


def main():
    """Run administrative tasks."""
    os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'config.settings')
    try:
        from django.core.management import execute_from_command_line
    except ImportError as exc:
        raise ImportError(
            "Couldn't import Django. Are you sure it's installed and "
            "available on your PYTHONPATH environment variable? Did you "
            "forget to activate a virtual environment?"
        ) from exc
    execute_from_command_line(sys.argv)


if __name__ == '__main__':
    main()
```

关键点：
- 设置 `DJANGO_SETTINGS_MODULE=config.settings`，指向 `config/settings.py`
- 通过 `uv run python manage.py <command>` 执行

## 3. config/settings.py（核心配置节选）

```python
from pathlib import Path

BASE_DIR = Path(__file__).resolve().parent.parent

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
]

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}

STATIC_URL = 'static/'
DEFAULT_AUTO_FIELD = 'django.db.models.BigAutoField'
```

- 默认 SQLite 数据库
- BigAutoField 作为主键类型（支持更多数据量）
- 后续按需添加自定义 App

## 4. config/urls.py（根路由）

```python
from django.contrib import admin
from django.urls import path

urlpatterns = [
    path('admin/', admin.site.urls),
]
```

- 默认仅注册 Admin 路由
- 后续添加博客相关 URL 配置

## 5. 数据库初始化

```bash
python manage.py migrate
```

创建了以下数据表：
- `auth_user` / `auth_group` / `auth_permission` — 用户认证
- `django_admin_log` — Admin 操作日志
- `django_content_type` — 内容类型注册
- `django_session` — 会话管理

## 6. 创建并注册blog app

```bash
python manage.py startapp blog
```

在 `config/settings.py` 中添加：

```python
INSTALLED_APPS = [
    # ... 其他已安装应用 ...
    'blog',
]
```
