### Tag（标签）

```python

    name = models.CharField(max_length=100)
```

- 只有 `name` 一个字段，用作文章标签
- 与 Post 为多对多关系（ManyToManyField）

### Post（文章）

```python
class Post(models.Model):
    title = models.CharField(max_length=70)
    body = models.TextField()
    created_time = models.DateTimeField()
    modified_time = models.DateTimeField()
    excerpt = models.CharField(max_length=200, blank=True)
    category = models.ForeignKey(Category, on_delete=models.CASCADE)
    tags = models.ManyToManyField(Tag, blank=True)
    author = models.ForeignKey(User, on_delete=models.CASCADE)
```

| 字段 | 类型 | 说明 |
|------|------|------|
| title | CharField(70) | 文章标题 |
| body | TextField | 文章正文，大段文本使用 TextField |
| created_time | DateTimeField | 创建时间 |
| modified_time | DateTimeField | 最后修改时间 |
| excerpt | CharField(200, blank=True) | 文章摘要，允许为空 |
| category | ForeignKey → Category | 所属分类，级联删除 |
| tags | ManyToManyField → Tag | 关联标签，允许为空 |
| author | ForeignKey → User | 作者，关联 django auth.User，级联删除 |

## 13. 模型管理界面与字段中文化（commit c40bfcf）

### 模型 Meta 类与字段 verbose_name

```python
from django.utils import timezone

class Category(models.Model):
    name = models.CharField(max_length=100)

    class Meta:
        verbose_name = '分类'
        verbose_name_plural = verbose_name


class Tag(models.Model):
    name = models.CharField(max_length=100)

    class Meta:
        verbose_name = '标签'
        verbose_name_plural = verbose_name


class Post(models.Model):
    title = models.CharField("标题", max_length=70)
    body = models.TextField("正文")
    created_time = models.DateTimeField("创建时间", default=timezone.now)
    modified_time = models.DateTimeField("修改时间")
    excerpt = models.CharField("摘要", max_length=200, blank=True)
    category = models.ForeignKey(Category, verbose_name="分类", on_delete=models.CASCADE)
    tags = models.ManyToManyField(Tag, verbose_name="标签", blank=True)
    author = models.ForeignKey(User, verbose_name="作者", on_delete=models.CASCADE)

    def save(self, *args, **kwargs):
        self.modified_time = timezone.now()
        super().save(*args, **kwargs)

    class Meta:
        verbose_name = '文章'
        verbose_name_plural = verbose_name
```

| 变更 | 说明 |
|------|------|
| `verbose_name` | 每个 Model 添加 Meta 类，Admin 后台显示中文名 |
| 字段第一参数 | 全部改为中文名称字符串，如 `"标题"`、`"正文"` |
| `created_time` | 新增 `default=timezone.now`，创建时自动填充 |
| `save()` 方法 | 重写 `save()`，每次保存自动更新 `modified_time` |

### Admin 后台注册（blog/admin.py）

```python
from django.contrib import admin
from .models import Category, Tag, Post

class PostAdmin(admin.ModelAdmin):
    list_display = ['title', 'created_time', 'modified_time', 'category', 'author']
    fields = ['title', 'body', 'excerpt', 'category', 'tags']

    def save_model(self, request, obj, form, change):
```

## 16. 文章目录支持（commit 85aeea9）

### 修复 markdown.markdown → markdown.Markdown（bug 修复）

```python
import re
from django.utils.text import slugify
from markdown.extensions.toc import TocExtension

def detail(request, pk):
    post = get_object_or_404(Post, pk=pk)
    md = markdown.Markdown(extensions=[
        'markdown.extensions.extra',
        'markdown.extensions.codehilite',
        TocExtension(slugify=slugify),
    ])
    post.body = md.convert(post.body)

    # 提取目录结构
    m = re.search(r'<div class="toc">\s*<ul>(.*)</ul>\s*</div>', md.toc, re.S)
    post.toc = m.group(1) if m is not None else ''

    return render(request, 'blog/detail.html', context={'post': post})
```

`md.toc` 由 `TocExtension` 注入到 Markdown 实例中，包含完整的 HTML 目录。

## 18. 摘要自动生成（commit 0dbb1c1）

### Post 模型的 save 方法增强（blog/models.py）

```python
import markdown
from django.utils.html import strip_tags

class Post(models.Model):
    # ... 字段定义 ...

    def save(self, *args, **kwargs):
        self.modified_time = timezone.now()

        # 实例化 Markdown，去掉 TocExtension（摘要不需要目录）
        md = markdown.Markdown(extensions=[
            'markdown.extensions.extra',
            'markdown.extensions.codehilite',
        ])

        # Markdown → HTML → 去除标签 → 取前 54 个字符作为摘要
        self.excerpt = strip_tags(md.convert(self.body))[:54]

        super().save(*args, **kwargs)
```

**流程**：`body (Markdown)` → `md.convert()` → `HTML` → `strip_tags()` → `纯文本[:54]` → `excerpt`

### 首页模板

- 自动生成的摘要末尾追加 `...` 省略号

### 注册模板标签库（blog/templatetags/blog_extras.py）

```python
from django import template
from ..models import Post, Category, Tag

register = template.Library()


@register.inclusion_tag('blog/inclusions/_recent_posts.html', takes_context=True)
def show_recent_posts(context, num=5):
    return {
        'recent_post_list': Post.objects.all().order_by('-created_time')[:num],
    }


@register.inclusion_tag('blog/inclusions/_archives.html', takes_context=True)
def show_archives(context):
    return {
        'date_list': Post.objects.dates('created_time', 'month', order='DESC'),
    }


@register.inclusion_tag('blog/inclusions/_categories.html', takes_context=True)
def show_categories(context):
    return {
        'category_list': Category.objects.all(),
    }


@register.inclusion_tag('blog/inclusions/_tags.html', takes_context=True)
def show_tags(context):
    return {
        'tag_list': Tag.objects.all(),
    }
```

| 标签 | 功能 | 数据来源 |
|------|------|---------|
| `{% show_recent_posts %}` | 最新文章（默认 5 篇） | `Post.objects.all()[:5]` |
| `{% show_archives %}` | 按月归档 | `Post.objects.dates('created_time', 'month')` |
| `{% show_categories %}` | 全部分类 | `Category.objects.all()` |
| `{% show_tags %}` | 全部标签 | `Tag.objects.all()` |

### base.html 使用


```python
from .models import Post, Category, Tag   # 补充导入 Category 和 Tag


def archive(request, year, month):
    """按年月归档：过滤指定年月的文章"""
    post_list = Post.objects.filter(
        created_time__year=year,
        created_time__month=month
    ).order_by('-created_time')
    return render(request, 'blog/index.html', context={'post_list': post_list})


def category(request, pk):
    """按分类筛选：过滤指定分类的文章"""
    cate = get_object_or_404(Category, pk=pk)
    post_list = Post.objects.filter(category=cate).order_by('-created_time')
    return render(request, 'blog/index.html', context={'post_list': post_list})


def tag(request, pk):
    """按标签筛选：过滤指定标签的文章"""
    t = get_object_or_404(Tag, pk=pk)
    post_list = Post.objects.filter(tags=t).order_by('-created_time')
    return render(request, 'blog/index.html', context={'post_list': post_list})
```

```python
from django.views.decorators.http import require_POST
from .forms import CommentForm
from django.contrib import messages


@require_POST
def comment(request, post_pk):
    post = get_object_or_404(Post, pk=post_pk)
    form = CommentForm(request.POST)

    if form.is_valid():
        # commit=False：先生成实例但不保存，以便关联 post
        comment = form.save(commit=False)
        comment.post = post
        comment.save()
        messages.add_message(request, messages.SUCCESS, '评论发表成功！', extra_tags='success')
        return redirect(post)  # 重定向到文章详情页

    # 表单不合法 → 渲染预览页显示错误
    context = {'post': post, 'form': form}
    messages.add_message(request, messages.ERROR, '评论发表失败！请修改表单中的错误后重新提交。', extra_tags='danger')
    return render(request, 'comments/preview.html', context=context)
```

2. 获取被评论的文章 post
3. 用 POST 数据构造 CommentForm
4. `form.is_valid()` → 合法则 save(commit=False) 关联 post 后保存，成功后 redirect(post)
5. 不合法则渲染 preview.html 显示错误信息

```python
class Post(models.Model):
```

```python
class PostDetailView(DetailView):
    model = Post
    template_name = 'blog/detail.html'
    context_object_name = 'post'

    def get(self, request, *args, **kwargs):
        response = super().get(request, *args, **kwargs)
        self.object.increase_views()   # 每次访问阅读量 +1
        return response

    def get_object(self, queryset=None):
        post = super().get_object(queryset=None)
        # Markdown 渲染 + 目录提取
        md = markdown.Markdown(extensions=[
            'markdown.extensions.extra',
            'markdown.extensions.codehilite',
            TocExtension(slugify=slugify),
        ])
        post.body = md.convert(post.body)
        m = re.search(r'<div class="toc">\s*<ul>(.*)</ul>\s*</div>', md.toc, re.S)
        post.toc = m.group(1) if m is not None else ''
        return post
```

| 覆写方法 | 作用 |
|---------|------|
| `get()` | 先调用父类获取响应，再调用 `increase_views()` 增加阅读量 |
| `get_object()` | 获取文章实例后先渲染 Markdown + 提取目录，再返回 |

### 归档/分类/标签视图（继承 IndexView）

```python
class ArchiveView(IndexView):
    def get_queryset(self):
        year = self.kwargs.get('year')
        month = self.kwargs.get('month')
        return super().get_queryset().filter(
            created_time__year=year, created_time__month=month)

class CategoryView(IndexView):
    def get_queryset(self):
        cate = get_object_or_404(Category, pk=self.kwargs.get('pk'))
        return super().get_queryset().filter(category=cate)

class TagView(IndexView):
    def get_queryset(self):
        t = get_object_or_404(Tag, pk=self.kwargs.get('pk'))
        return super().get_queryset().filter(tags=t)
```

三个视图都继承 `IndexView`，只覆写 `get_queryset()` 方法添加过滤条件，重复代码降为 0。

### 类视图优势

| 对比 | 函数视图 | 类视图 |
|------|---------|--------|
| 代码量 | 80+ 行，视图间重复逻辑多 | 54 行，Archive/Category/Tag 复用 IndexView |
| 可扩展性 | 需每个视图独立实现 | get_queryset() 覆写即可 |
| 阅读量逻辑 | 耦合在视图函数中 | 通过 get() 生命周期方法解耦 |

## 29. 分页功能（commit f59dfe2）

### IndexView 添加分页

```python
# blog/views.py
class IndexView(ListView):
    model = Post
    template_name = 'blog/index.html'
    context_object_name = 'post_list'
    paginate_by = 10   # 每页 10 篇文章
```

Django 的 `ListView` 内置了分页支持，只需设置 `paginate_by` 即可启用。模板中自动获得两个上下文变量：
- `page_obj` — 当前页的对象（含 `has_previous`、`has_next`、`number` 等属性）
- `paginator` — 分页器（含 `num_pages` 总页数）

### 模板分页导航

```html
{% if is_paginated %}
  <div class="pagination-simple">
    {% if page_obj.has_previous %}
      <a href="?page={{ page_obj.previous_page_number }}">上一页</a>
    {% endif %}
    <span class="current">
      第 {{ page_obj.number }} 页 / 共 {{ paginator.num_pages }} 页
    </span>
    {% if page_obj.has_next %}
      <a href="?page={{ page_obj.next_page_number }}">下一页</a>
    {% endif %}
  </div>
{% endif %}
```

| 模板变量 | 说明 |
|---------|------|
| `is_paginated` | 是否启用了分页（超过一页时为 True） |
| `page_obj.has_previous` | 是否有上一页 |
| `page_obj.previous_page_number` | 上一页页码 |
| `page_obj.number` | 当前页码 |
| `page_obj.has_next` | 是否有下一页 |
| `page_obj.next_page_number` | 下一页页码 |
| `paginator.num_pages` | 总页数 |

- 当文章不足 `paginate_by` 时，`is_paginated` 为 False，分页导航自动隐藏
- 首页、归档、分类、标签视图都继承 `IndexView`，全部自动获得分页能力
