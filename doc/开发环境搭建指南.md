# 开发环境搭建指南

> 记录时间：2026-06-20

## 前置条件

- Python >= 3.14
- [uv](https://docs.astral.sh/uv/)（Python 包管理器）
- Git

## 初始化步骤

### 1. 克隆项目

```bash
git clone <repo-url>
cd rance-blog
```

### 2. 安装依赖

```bash
uv sync
```

这会根据 `pyproject.toml` 和 `uv.lock` 自动创建虚拟环境并安装所有依赖。

### 3. 生成数据库迁移

```bash
uv run python manage.py makemigrations
```

### 4. 数据库迁移

```bash
uv run python manage.py migrate
```

### 5. 创建超级用户

```bash
uv run python manage.py createsuperuser
```

### 6. 启动开发服务器

```bash
uv run python manage.py runserver
```

默认访问 `http://127.0.0.1:8000/`，首页显示 "欢迎来到我的博客首页"。

## 配置说明

项目使用 split settings（配置分离）模式，开发环境与生产环境配置分开管理：

```
config/settings/
├── common.py      # 公共配置（数据库、App注册、中间件等）
├── local.py       # 开发配置（DEBUG=True，开发密钥）
└── production.py  # 生产配置（DEBUG=False，环境变量密钥）
```

- 开发时 `manage.py` 默认加载 `config.settings.local`
- 部署时 `wsgi.py` 默认加载 `config.settings.production`
