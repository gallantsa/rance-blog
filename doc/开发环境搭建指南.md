# 开发环境搭建指南

> 记录时间：2026-06-20

## 前置条件

- Python >= 3.14
- [uv](https://docs.astral.sh/uv/)（Python 包管理器）
- Git

## 初始化步骤

### 1. 克隆项目

```bash
git clone <repo-url>
cd rance-blog
```

### 2. 安装依赖

```bash
uv sync
```

这会根据 `pyproject.toml` 和 `uv.lock` 自动创建虚拟环境并安装所有依赖。

### 3. 数据库迁移

```bash
uv run python manage.py migrate
```

### 4. 创建超级用户

```bash
uv run python manage.py createsuperuser
```

### 5. 启动开发服务器

```bash
uv run python manage.py runserver
```

默认访问 `http://127.0.0.1:8000/`

## 常用命令速查

| 操作 | 命令 |
|------|------|
| 安装新依赖 | `uv add <package>` |
| 删除依赖 | `uv remove <package>` |
| 同步依赖 | `uv sync` |
| 创建 Django app | `uv run python manage.py startapp <app_name>` |
| 创建数据库迁移 | `uv run python manage.py makemigrations` |
| 执行迁移 | `uv run python manage.py migrate` |
| 打开 Django shell | `uv run python manage.py shell` |
| 收集静态文件 | `uv run python manage.py collectstatic` |
| 运行测试 | `uv run python manage.py test` |
| Django 版本检查 | `uv run python -m django --version` |

## 已完成的初始化操作

1. ✅ `uv init` / `uv sync` — 初始化 Python 项目及虚拟环境
2. ✅ `pyproject.toml` 配置 — 添加 `django>=5.2,<6.0` 依赖
3. ✅ `django-admin startproject config .` — 创建 Django 项目
4. ✅ `python manage.py migrate` — 初始化默认 SQLite 数据库
