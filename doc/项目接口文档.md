# 项目接口文档

> 记录时间：2026-06-20

## URL 路由一览

### 根路由（config/urls.py）

| 路径 | 目标 | 说明 |
|------|------|------|
| `admin/` | `django.contrib.admin.site.urls` | Django Admin 后台 |
| `` (空) | `blog.urls` | 博客前台 |

### blog 应用路由（blog/urls.py）

命名空间：`blog`

| URL | 视图 | name | 说明 |
|-----|------|------|------|
| `` | `views.index` | `blog:index` | 博客首页，文章列表 |
| `post/<int:pk>/` | `views.detail` | `blog:detail` | 文章详情页 |

## 视图函数

### blog/views.py

#### `index(request)`
- **功能**：渲染博客首页，显示文章列表
- **模板**：`blog/index.html`
- **上下文**：
  ```python
  {
      'post_list': Post  # 按 created_time 倒序排列的所有文章
  }
  ```

#### `detail(request, pk)`
- **功能**：渲染单篇文章详情页
- **URL 参数**：`pk` — 文章主键（int）
- **模板**：`blog/detail.html`
- **上下文**：
  ```python
  {
      'post': Post  # 匹配主键的单个文章对象，不存在返回 404
  }
  ```

## 模板结构

| 模板文件 | 说明 |
|---------|------|
| `templates/base.html` | 基础模板，定义导航栏、侧边栏、页脚骨架 |
| `templates/blog/index.html` | 首页，继承 base.html，填充 `{% block main %}` |
| `templates/blog/detail.html` | 详情页，继承 base.html，填充 `{% block main %}` 和 `{% block toc %}` |
