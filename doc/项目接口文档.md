## 视图类

### blog/views.py

全部已重构为 Django 通用类视图。

#### `IndexView`（首页）
- **继承**：`ListView`
- **模型**：`Post`
- **模板**：`blog/index.html`
- **分页**：`paginate_by = 10`（每页 10 篇，自动通过 `?page=N` 参数翻页）
- **上下文**：`post_list` — 当前页文章；`page_obj` / `paginator` — 分页对象
- **子类复用**：ArchiveView、CategoryView、TagView 均继承此类，自动获得分页

#### `PostDetailView`（文章详情）
- **继承**：`DetailView`
- **模型**：`Post`
- **模板**：`blog/detail.html`
- **上下文**：`post` — 文章对象（body 已渲染为 HTML，含 toc 目录）
- **内部处理流程**：
  1. `get_object()` — 获取文章 → Markdown 渲染 body → 提取目录存入 `post.toc`
  2. `get()` — 调用父类获取响应 → `increase_views()` 阅读量 +1
- **模板注意**：`{{ post.body | safe }}`（Markdown 渲染的 HTML 需 safe 过滤器）

#### `ArchiveView`（归档）
- **继承**：`IndexView`
- **功能**：按年月筛选文章
- **URL 参数**：`year` (int)、`month` (int)
- **覆写**：`get_queryset()` — `filter(created_time__year=year, created_time__month=month)`

#### `CategoryView`（分类）
- **继承**：`IndexView`
- **功能**：按分类筛选文章
- **URL 参数**：`pk` — 分类主键（int）
- **URL 路径**：`category/<int:pk>/`
- **覆写**：`get_queryset()` — `filter(category=get_object_or_404(Category, pk=pk))`

#### `TagView`（标签）
- **继承**：`IndexView`
- **功能**：按标签筛选文章
- **URL 参数**：`pk` — 标签主键（int）
- **覆写**：`get_queryset()` — `filter(tags=get_object_or_404(Tag, pk=pk))`

### blog/views.py（函数视图）

#### `search(request)`
- **功能**：全文搜索文章标题和正文
- **URL 参数**：`q` — 搜索关键词（GET）
- **模板**：`blog/index.html`（复用首页模板）
- **ORM 查询**：`Post.objects.filter(Q(title__icontains=q) \| Q(body__icontains=q))`
- **行为**：
  - 有关键词 → 返回匹配的文章列表
  - 无关键词 → 错误消息 + 重定向到首页

### comments/views.py

#### `comment(request, post_pk)`
- **功能**：处理评论表单提交
- **URL 参数**：`post_pk` — 文章主键（int）
- **HTTP 方法**：仅 POST（`@require_POST` 装饰器）
- **模板**：
  - 成功 → `redirect(post)`（重定向到文章详情页）
  - 失败 → `comments/preview.html`
- **处理流程**：
  1. `get_object_or_404(Post, pk=post_pk)` 获取被评论文章
  2. `CommentForm(request.POST)` 绑定表单数据
  3. 合法 → `form.save(commit=False)` 关联 post → `comment.save()` → 成功消息 → `redirect(post)`
  4. 不合法 → 错误消息 → 渲染 `preview.html`（含表单错误信息）
- **Django Messages**：成功 → `messages.SUCCESS`（alert-success）；失败 → `messages.ERROR`（alert-danger）

## 模板结构

| 模板文件 | 说明 |
|---------|------|
| `templates/base.html` | 基础模板，定义导航栏、侧边栏（动态小部件）、页脚骨架 |
| `templates/blog/index.html` | 首页，继承 base.html，填充 `{% block main %}` |
| `templates/blog/detail.html` | 详情页，继承 base.html，填充 `{% block main %}` 和 `{% block toc %}` |
| `templates/blog/inclusions/_recent_posts.html` | 最新文章小部件 |
| `templates/blog/inclusions/_archives.html` | 按月归档小部件 |
| `templates/blog/inclusions/_categories.html` | 全部分类小部件 |
| `templates/blog/inclusions/_tags.html` | 标签云小部件 |
| `templates/comments/preview.html` | 评论提交失败预览页（继承 base.html，加载 comments_extras） |
| `templates/comments/inclusions/_form.html` | 评论表单（name/email/url/text + csrf_token） |
| `templates/comments/inclusions/_list.html` | 评论列表（含条数统计、comments_filter、空状态） |
