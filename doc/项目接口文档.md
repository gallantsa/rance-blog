## URL 路由一览

### 根路由（config/urls.py）

| 路径 | 目标 | 说明 |

| `admin/` | `django.contrib.admin.site.urls` | Django Admin 后台 |
| `` (空) | `blog.urls` | 博客前台 |

### blog 应用路由（blog/urls.py）

命名空间：`blog`

| URL | 视图 | name | 说明 |
|-----|------|------|------|
| `` | `views.index` | `blog:index` | 博客首页，文章列表 |
| `post/<int:pk>/` | `views.detail` | `blog:detail` | 文章详情页 |
| `archives/<int:year>/<int:month>/` | `views.archive` | `blog:archive` | 按年月归档 |
| `categories/<int:pk>/` | `views.category` | `blog:category` | 按分类筛选 |
| `tags/<int:pk>/` | `views.tag` | `blog:tag` | 按标签筛选 |

## 视图函数

### blog/views.py

#### `index(request)`
- **功能**：渲染博客首页，显示文章列表
- **模板**：`blog/index.html`
- **上下文**：
  ```python
  {
      'post_list': Post  # 按 created_time 倒序排列的所有文章
  }
  ```

#### `detail(request, pk)`
- **功能**：渲染单篇文章详情页，支持 Markdown 渲染、代码高亮、自动目录
- **URL 参数**：`pk` — 文章主键（int）
- **模板**：`blog/detail.html`
- **内部处理流程**：
  1. 通过 `get_object_or_404(Post, pk=pk)` 获取文章
  2. 使用 `markdown.Markdown()` 将 `post.body` 从 Markdown 转换为 HTML
  3. 通过 `TocExtension(slugify=slugify)` 生成文章目录
  4. 从 `md.toc` 提取 `<ul>` 目录结构存入 `post.toc`
- **上下文**：
  ```python
  {
      'post': Post,      # 文章对象（body 已被渲染为 HTML）
      # post.body  — Markdown → HTML（模板中需 safe 过滤器）
      # post.toc   — 文章目录 HTML 片段（可能为空）
  }
  ```

#### `archive(request, year, month)`
- **功能**：按年月归档，筛选指定年月的文章
- **URL 参数**：`year` (int)、`month` (int)
- **模板**：`blog/index.html`（复用首页模板）
- **ORM 查询**：`Post.objects.filter(created_time__year=year, created_time__month=month)`
- **上下文**：
  ```python
  { 'post_list': Post  }  # 筛选后的文章列表
  ```

#### `category(request, pk)`
- **功能**：按分类筛选文章
- **URL 参数**：`pk` — 分类主键（int）
- **模板**：`blog/index.html`（复用首页模板）
- **ORM 查询**：`Post.objects.filter(category=get_object_or_404(Category, pk=pk))`
- **上下文**：
  ```python
  { 'post_list': Post  }
  ```

#### `tag(request, pk)`
- **功能**：按标签筛选文章
- **URL 参数**：`pk` — 标签主键（int）
- **模板**：`blog/index.html`（复用首页模板）
- **ORM 查询**：`Post.objects.filter(tags=get_object_or_404(Tag, pk=pk))`
- **上下文**：
  ```python
  { 'post_list': Post  }
  ```

## 模板结构

| 模板文件 | 说明 |
|---------|------|
| `templates/base.html` | 基础模板，定义导航栏、侧边栏（动态小部件）、页脚骨架 |
| `templates/blog/index.html` | 首页，继承 base.html，填充 `{% block main %}` |
| `templates/blog/detail.html` | 详情页，继承 base.html，填充 `{% block main %}` 和 `{% block toc %}` |
| `templates/blog/inclusions/_recent_posts.html` | 最新文章小部件 |
| `templates/blog/inclusions/_archives.html` | 按月归档小部件 |
| `templates/blog/inclusions/_categories.html` | 全部分类小部件 |
| `templates/blog/inclusions/_tags.html` | 标签云小部件 |
