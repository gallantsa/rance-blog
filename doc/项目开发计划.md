### 项目结构
```
rance-blog/
├── config/              # Django 项目配置
│   ├── __init__.py
│   ├── asgi.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── doc/                 # 项目文档
├── manage.py            # Django 管理入口
├── pyproject.toml       # 项目配置与依赖
├── uv.lock              # 依赖锁定文件
├── db.sqlite3           # 开发数据库
└── .gitignore
```

### 技术栈
- Python 3.14 + Django 5.2 LTS
- uv 包管理
- SQLite（开发环境）
