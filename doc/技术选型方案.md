# 技术选型方案

> 记录时间：2026-06-20

## 当前选型

| 维度 | 选型 | 版本 | 说明 |
|------|------|------|------|
| **编程语言** | Python | 3.14.6 | 项目最低要求 Python >= 3.14 |
| **Web 框架** | Django | 5.2.15 | LTS 系列最新版，长期支持 |
| **包管理工具** | uv | 0.11.21 | 高性能 Python 包管理器 |
| **数据库** | SQLite（开发） | — | 默认配置，后续可按需切换为 PostgreSQL |

## 选型理由

### Django 5.2
- 成熟稳定的 Web 框架，ORM、Admin、认证、模板引擎开箱即用
- 5.2 为 LTS 版本，长期维护，社区生态成熟
- 适合博客类内容管理系统的快速开发

### uv
- 相比 pip/poetry 安装速度快 10-100 倍
- 自动生成 `uv.lock` 锁定依赖版本
- 原生支持 pyproject.toml，无需额外配置
