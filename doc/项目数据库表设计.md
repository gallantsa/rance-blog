## 表结构详情

### blog_category（分类）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BigAutoField | PK, 自增 | 主键 ID |
| name | CharField(100) | NOT NULL | 分类名称 |

SQL:
```sql
CREATE TABLE "blog_category" (
    "id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
    "name" varchar(100) NOT NULL
);
```

### blog_tag（标签）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BigAutoField | PK, 自增 | 主键 ID |
| name | CharField(100) | NOT NULL | 标签名称 |

SQL:
```sql
CREATE TABLE "blog_tag" (
    "id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
    "name" varchar(100) NOT NULL
);
```

### blog_post（文章）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BigAutoField | PK, 自增 | 主键 ID |
| title | CharField(70) | NOT NULL | 文章标题 |
| body | TextField | NOT NULL | 文章正文 |
| created_time | DateTimeField | NOT NULL | 创建时间 |
| modified_time | DateTimeField | NOT NULL | 最后修改时间 |
| excerpt | CharField(200) | NULLABLE (`blank=True`) | 文章摘要 |
| views | PositiveIntegerField | NOT NULL, default=0 | 阅读量，每次访问 +1 |
| category_id | BigInt | FK → `blog_category.id` | 所属分类 |
| author_id | BigInt | FK → `auth_user.id` | 作者 |

SQL:
```sql
CREATE TABLE "blog_post" (
    "id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
    "title" varchar(70) NOT NULL,
    "body" text NOT NULL,
    "created_time" datetime NOT NULL,
    "modified_time" datetime NOT NULL,
    "excerpt" varchar(200) NOT NULL,
    "category_id" bigint NOT NULL REFERENCES "blog_category" ("id") DEFERRABLE INITIALLY DEFERRED,
    "author_id" bigint NOT NULL REFERENCES "auth_user" ("id") DEFERRABLE INITIALLY DEFERRED
);
CREATE INDEX "blog_post_category_id_..." ON "blog_post" ("category_id");
CREATE INDEX "blog_post_author_id_..." ON "blog_post" ("author_id");
```

### blog_post_tags（文章-标签关联表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BigAutoField | PK, 自增 | 主键 ID |
| post_id | BigInt | FK → `blog_post.id` | 文章 ID |
| tag_id | BigInt | FK → `blog_tag.id` | 标签 ID |

SQL:
```sql
CREATE TABLE "blog_post_tags" (
    "id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
    "post_id" bigint NOT NULL REFERENCES "blog_post" ("id") DEFERRABLE INITIALLY DEFERRED,
    "tag_id" bigint NOT NULL REFERENCES "blog_tag" ("id") DEFERRABLE INITIALLY DEFERRED
);
CREATE UNIQUE INDEX "blog_post_tags_post_id_tag_id_uniq" ON "blog_post_tags" ("post_id", "tag_id");
```

## 关联关系

| 关系 | 类型 | 说明 |
|------|------|------|
| Post → Category | 多对一 (ForeignKey) | 一篇文章属于一个分类，一个分类有多篇文章；分类删除时级联删除文章 |
| Post → Tag | 多对多 (ManyToManyField) | 一篇文章可以有多个标签，一个标签可被多篇文章使用；可选（`blank=True`） |
| Post → User | 多对一 (ForeignKey) | 一篇文章有一个作者，一个作者可写多篇文章；作者删除时级联删除文章 |

## 备注

- 使用 Django 默认的 `BigAutoField` 作为主键，支持更大数据量
- 迁移文件由 `python manage.py makemigrations` 自动生成
- 数据库文件路径为 `database/db.sqlite3`（位于项目根目录的 database/ 子目录下，便于 Docker volume 挂载持久化）
- 生产环境建议切换为 PostgreSQL

## Comment 模型（评论模块）

### 表结构

表名：`comments_comment`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BigAutoField | PK, 自增 | 主键 ID |
| name | CharField(50) | NOT NULL | 评论者名字 |
| email | EmailField | NOT NULL | 邮箱 |
| url | URLField | NULLABLE (`blank=True`) | 个人网址 |
| text | TextField | NOT NULL | 评论正文 |
| created_time | DateTimeField | NOT NULL, default=timezone.now | 创建时间 |
| post_id | BigInt | FK → `blog_post.id` | 所属文章 |

SQL:
```sql
CREATE TABLE "comments_comment" (
    "id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
    "name" varchar(50) NOT NULL,
    "email" varchar(254) NOT NULL,
    "url" varchar(200) NOT NULL,
    "text" text NOT NULL,
    "created_time" datetime NOT NULL,
    "post_id" bigint NOT NULL REFERENCES "blog_post" ("id") DEFERRABLE INITIALLY DEFERRED
);
CREATE INDEX "comments_comment_post_id" ON "comments_comment" ("post_id");
```

### save() 方法

每次保存 Post 时自动更新 `modified_time`，并自动生成摘要在 save() 方法中增强为三合一逻辑：

```python
import markdown
from django.utils.html import strip_tags

def save(self, *args, **kwargs):
    self.modified_time = timezone.now()

    # 自动摘要生成：Markdown → HTML → 去标签 → 取前 54 字
    md = markdown.Markdown(extensions=[
        'markdown.extensions.extra',
        'markdown.extensions.codehilite',
    ])
    self.excerpt = strip_tags(md.convert(self.body))[:54]

    super().save(*args, **kwargs)
```

| 操作 | 说明 |
|------|------|
| `modified_time = timezone.now()` | 每次保存更新修改时间 |
| `md.convert(self.body)` | 将 Markdown 正文渲染为 HTML |
| `strip_tags(...)` | 去除全部 HTML 标签，得到纯文本 |
| `[:54]` | 取前 54 个字符作为摘要 |

### get_absolute_url 方法

Post 模型实现了获取详情页 URL 的方法：

```python
def get_absolute_url(self):
    return reverse('blog:detail', kwargs={'pk': self.pk})
```

- 使用 URL 命名空间 `blog:detail`，对应路由 `/post/<pk>/`
- 模板中通过 `{{ post.get_absolute_url }}` 直接生成链接
